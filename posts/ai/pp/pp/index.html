在 Ubuntu 上使用 conda 创建一个包含 CUDA 11.2 和 PaddlePaddle-GPU 2.6 的环境，你可以按照以下步骤操作。首先，请确保你已经安装了 miniconda 或 anaconda。如果还没有安装，可以从 Miniconda 或 Anaconda 官网下载安装包并进行安装。

步骤 1: 创建一个新的 Conda 环境

打开终端，并创建一个新的 Conda 环境，命名为 paddle_cuda112（你可以根据需要更改环境名）。

1

conda create -n paddle_cuda112 python=3.8

这里我们选择了 Python 3.8，因为 PaddlePaddle 2.6 支持这个版本。你可以根据需要选择其他支持的 Python 版本。

步骤 2: 激活环境

激活你刚刚创建的环境：

1

conda activate paddle_cuda112

步骤 3: 安装 CUDA 11.6

Conda 默认不支持直接安装特定版本的 CUDA。不过，你可以通过 NVIDIA 提供的 Conda 渠道安装 CUDA Toolkit 和 cuDNN。不过，需要注意的是，Conda 提供的 CUDA 版本可能有限，不一定总是包含你需要的特定版本。如果 Conda 中没有 CUDA 11.2，你可能需要从 NVIDIA 官网手动下载并安装 CUDA 11.2。

但首先，尝试使用 Conda 安装 CUDA Toolkit（如果可用）：

1

conda install -c nvidia cuda-toolkit=11.6

如果 Conda 中没有 CUDA 11.2，你需要从 NVIDIA CUDA Toolkit Archive 下载并手动安装。

步骤 4: 安装 cuDNN

类似地，尝试使用 Conda 安装 cuDNN（如果可用）。注意，cuDNN 的版本需要与 CUDA 版本匹配。

1

conda install -c nvidia cudnn=8.1.0  # cuDNN 8.1.0 兼容 CUDA 11.2

如果 Conda 中没有所需版本的 cuDNN，你也需要从 NVIDIA cuDNN Archive 下载并手动安装。

步骤 5: 安装 PaddlePaddle-GPU

使用 Conda 安装 PaddlePaddle-GPU 2.6。由于 PaddlePaddle 官方提供了 Conda 安装包，你可以直接从官方渠道安装：

1
2
3

bash复制代码

conda install paddlepaddle-gpu=2.6.0 cudatoolkit=11.2 -c paddlepaddle

注意：这里 -c paddlepaddle 指定从 PaddlePaddle 官方 Conda 渠道安装。

步骤 6: 验证安装

安装完成后，你可以通过以下命令验证 PaddlePaddle-GPU 是否正确安装，并检查 CUDA 是否可用：

1
2
3

import paddle
print(paddle.__version__)  # 应该输出 2.6.0
print(paddle.utils.run_check())  # 检查 PaddlePaddle 是否正确安装，包括 GPU 支持

如果一切正常，你应该会看到类似以下的输出，表明 PaddlePaddle-GPU 已正确安装并且 CUDA 可用：

1
2
3

复制代码

PaddlePaddle works well on your machine!

注意事项

  • 如果 Conda 中没有提供你需要的 CUDA 或 cuDNN 版本，请务必从 NVIDIA 官网下载并手动安装。
  • 确保你的 GPU 驱动与 CUDA 版本兼容。
  • 如果遇到任何问题，请检查 PaddlePaddle 和 CUDA 的官方文档，以及 Conda 的安装日志，以获取更多帮助。

这样，你就应该能够在 Ubuntu 上成功创建一个包含 CUDA 11.2 和 PaddlePaddle-GPU 2.6 的 Conda 环境了。
